# SCT_DA_1 - Excel Sales Dashboard

## 📌 Task Description
Use Microsoft Excel to clean and analyze the Superstore 
Sales dataset. Create a dashboard using Pivot Tables 
and Charts to visualize Total Sales, Profit by Category, 
and Sales Trends over time.

## 🛠️ Tools & Technologies
- Microsoft Excel
- Superstore Sales Dataset

## 📂 Dataset
Superstore Sales Dataset (train.csv)
- 9,800 rows
- 18 columns

## ✅ What Was Built
- Pivot Table 1 - Sales by Category
- Pivot Table 2 - Sales by Region
- Pivot Table 3 - Monthly Sales Trend
- Bar Chart - Sales by Category
- Column Chart - Sales by Region
- Line Chart - Sales Trend Over Time
- Conditional Formatting on Sales column
- Slicers for Region and Category filtering
- Final Dashboard sheet

## 💡 Key Insights
- Technology is the top selling category at $827,455
- Sales grew consistently from 2015 to 2018
- Office Supplies has the lowest sales at $705,422
- West region has the highest sales
